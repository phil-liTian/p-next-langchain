## 目录说明

- **app/** - Next.js 13+ App Router 结构，包含所有页面和 API 路由
- **components/** - React 组件，分为通用组件和 UI 组件库
- **public/** - 静态资源文件
- **utils/** - 工具函数
- **data/** - 静态数据文件
